# TaskMaster - Task Management System

A full-stack task management application built with Node.js, Express, MongoDB, and vanilla JavaScript.

## Features

- User Authentication (Register/Login/Logout)
- Task CRUD Operations (Create, Read, Update, Delete)
- Task Filtering by Priority, Status, and Deadline
- Search Tasks by Title or Description
- Responsive Dashboard with Statistics
- Modern UI with Blue and White Theme
- JWT Authentication
- MongoDB Database
- RESTful API

## Technology Stack

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT Authentication
- bcrypt for password hashing
- Express Validator
- Helmet for security
- CORS

### Frontend
- HTML5
- CSS3 with responsive design
- Vanilla JavaScript
- Fetch API for HTTP requests
- LocalStorage for JWT token

### Testing
- Jest
- Supertest

### Deployment
- Backend: Fly.io
- Frontend: Vercel
